Uvod Tehnologije Primer

Šta je web i kako funkcioniše web sajt

Web predstavlja sistem povezanih stranica dostupnih putem interneta. Svaki web sajt ili veb sajt koristi HTML za strukturu i CSS za vizuelni izgled.

HTML CSS tehnologije za web i veb sajt

Za izradu svakog web sajta koriste se osnovne veb tehnologije.

  • HTML – definiše strukturu web sajta
  • CSS – oblikuje dizajn veb sajta
  • JavaScript – dodaje interaktivnost web stranici

Više o osnovama možeš pročitati na stranici šta je web.
