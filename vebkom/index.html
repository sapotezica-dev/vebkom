Uvod Tehnologije Primer

Šta je veb komunikacija?

Veb komunikacija predstavlja razmenu podataka između klijenta (najčešće web pregledača) i servera putem interneta. Najčešće se ostvaruje pomoću HTTP ili HTTPS protokola.

Osnovne tehnologije

  • HTML – struktura stranice
  • CSS – stil i izgled
  • JavaScript – interakcija i dinamika
  • HTTP/HTTPS – protokoli za komunikaciju
